# Коворкинг-Центр — Информационная система бронирования рабочих мест

Веб-приложение для автоматизации бронирования рабочих мест в коворкинг-пространствах.

## Рабочая группа и распределение ролей:

- **Backend-разработчик: Галиуллина А.Р.**  
  Разработка серверной части (Express.js), API, работа с базой данных, развертывание, тестирование и документация

- **Frontend-разработчик: Твеленёва А.А.**  
  Разработка клиентской части, интерфейса (React), документация


## Тематика проекта
Информационная система бронирования коворкинг-пространств.

## Объект автоматизации
Процесс бронирования рабочих мест в коворкинг-центрах, включая поиск, выбор, расчет стоимости и подтверждение бронирования.

## Параметры системы

### Постоянные:
- Адреса коворкинг-центров
- Перечень рабочих мест и инвентаря
- Учетные записи пользователей
- Базовые тарифы

### Изменяющиеся в реальном времени:
- Текущая занятость мест
- Статусы бронирований

### Изменяемые администратором:
- Список менеджеров и клиентов

### Изменяемые менеджером:
- Перечень коворкинг-центров и рабочих мест
- Инвентарь
- Расстановка мест на плане помещения
- Базовые тарифы
- Скидки
- Отмена бронирований

### Изменяемые клиентом:
- Бронирования клиента

## Концепция системы "Коворкинг-Центр"

**Цель системы** – автоматизация процесса бронирования рабочих мест в сети коворкинг-пространств для повышения удобства клиентов и эффективности использования площадей.

### Ключевые функции:
- Просмотр свободных рабочих мест в коворкинг-центрах на выбранную дату
- Расчет итоговой стоимости брони с учетом динамического ценообразования
- Оформление, подтверждение и отмена бронирований
- Управление инвентарем (наличием дополнительного оборудования)
- Расстановка мест на плане помещения
- Ведение личного кабинета пользователя с историей броней
